Privacy Policy

Effective date: May 7, 2026  ·  Workspace for Claude by Tidewell

Short version: We collect nothing. Your data is stored in your own browser and never leaves your device. We have no servers and no accounts.

1. Who we are

Workspace for Claude is a Chrome browser extension built and maintained by Tidewell. We are not affiliated with Anthropic, the makers of Claude. This policy describes how we handle data when you use the extension.

2. Data we collect

We do not collect any personal data. We have no servers, no database, and no way to receive data from your browser. Specifically, we never receive:

  • Your name, email address, or any account information
  • Your Claude conversation content
  • Your saved folders, prompts, or settings
  • Your IP address or location
  • Any usage analytics or behavioural data

3. Data stored on your device

The extension stores the following data locally in your browser using Chrome's chrome.storage.local API. This data exists only on your device:

  • Folders — names, nesting structure, and which chats are assigned
  • Prompts — titles, body text, variable definitions, and any attached files (stored as base64 in your browser)
  • Settings — keyboard shortcut preferences, insertion mode, and auto-send defaults
  • Onboarding state — whether you have completed the first-run tutorial
  • License cache — if you activate a Pro license, the result of the last validation check is cached for 24 hours (see Section 5)

You can delete all locally stored data at any time via Settings → Data → Clear all data inside the extension.

4. How we interact with Claude.ai

The extension injects a content script into claude.ai pages. This script:

  • Reads the current page URL to identify which conversation you are viewing
  • Injects UI elements (folder assign button, export button, prompt palette)
  • Inserts text into Claude's chat input when you select a prompt
  • Downloads the conversation DOM as a Markdown file when you click the export button

The extension does not record, log, transmit, or store any conversation content. It does not call any Claude or Anthropic API endpoints.

5. License validation (Pro users only)

If you choose to activate a Pro license, the license key you enter is sent to Gumroad's license validation API (api.gumroad.com) to verify that the key is valid. This is the only network request the extension makes to an external service, and it only happens when you click "Activate" in Settings or once per 24 hours to refresh the cached result.

The data sent to Gumroad is limited to your license key. Gumroad's handling of this request is governed by Gumroad's Privacy Policy.

6. No analytics or telemetry

We use no analytics tools, error tracking SDKs, or telemetry of any kind. We do not use Google Analytics, Mixpanel, Sentry, or similar services. If something breaks, users contact us directly.

7. Third-party services

The extension does not communicate with any third-party service except Gumroad (for Pro license validation, as described in Section 5). It does not load any external scripts, fonts, or assets.

8. Children's privacy

This extension is not directed at children under 13. We do not knowingly collect any data from children.

9. Changes to this policy

If we make material changes to this policy, we will update the effective date at the top of this page. Continued use of the extension after changes are posted constitutes your acceptance of the updated policy.
